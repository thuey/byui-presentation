Three Parts of a Web Page

  1. Structure (HTML): What content is on the page and how it is arranged.
  2. Styling (CSS): What the content looks like.
  3. Interactivity (JavaScript): How the user can interact with the page.
Completed JavaScript example.

Why React?

  1. Declarative vs Imperative: Describe what you want instead of how you want it done.
  2. Easy to re-use components: You can write code once and use it in a lot of places.
  3. React everywhere!: You can use React to write web apps, blogs, desktop apps, and mobile apps.
